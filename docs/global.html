removeObjectFromArray(array, field, value) → {Array|number}

Source:
  • removeObjectFromArray.js, line 3

Finds the first object in the array that matches the field/value pair passed in and removes it

If the array contains more than one object with the matching field/value pair, only the first object is removed.

Example
const arr = [{ name: 'joe', age: 20 }, { name: 'jill', age: 21 }]

removeObjectFromArray(arr, 'age', 21)

// => [{ name: 'joe', age: 20 }]
Parameters:
Name Type Description
array Array

an array of objects
field string

name of field/property evaluage
value *

value of the field/property evaluate
Returns:

  • If a match for the field/value pair is found, returns a new array with the specified object removed

    Type
    Array

  • If a match for field/value is not found, returns false.

    Type
    number

updateObjectInArray(array, field, value, newObject) → {Array|number}

Source:
  • updateObjectInArray.js, line 36

Finds the first object in the array that matches the field/value pair passed in and updates the object with properties contained in newObject

If the array contains more than one object with the matching field/value pair, only the first object found is updated.

Example
const arr = [{ name: 'joe', age: 20 }, { name: 'jill', age: 21 }]

updateObjectInArray(arr, 'name', 'joe', { name: 'joseph' })

// => [{ name: 'joseph', age: 20 }, { name: 'jill', age: 21 }]
